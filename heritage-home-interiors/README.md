# Heritage Home Interiors Website

A minimal-luxury, responsive marketing site for an interior design studio.

## Preview locally

- Open `index.html` in your browser (no build needed).

## Customize

- Replace Unsplash images with your own.
- Update copy in `index.html` to reflect your brand voice.
- Tweak colors and spacing in `styles.css`.

## Suggested pages/sections to add

- Case study pages per project
- Testimonials
- Process timeline
- Blog / Journal (optional)

## Deployment

- Static hosting options: GitHub Pages, Netlify, Vercel, Azure Static Web Apps.
